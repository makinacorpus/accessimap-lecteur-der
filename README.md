# Comment lancer et compiler les protos ?

## ELECTRON

Lancer l'app

```
electron .
```

Compiler l'app avec [electron packager](https://github.com/electron-userland/electron-packager)

```
electron-packager <sourcedir> <appname> --platform=<platform> --arch=<arch>
```
